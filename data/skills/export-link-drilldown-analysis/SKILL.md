# 出口环节下钻分析

本 Skill 用于分析出口环节中机构的时限表现，识别线路中哪些机构导致整体时限偏慢。

## 功能说明

1. 读取上级目录 `/data/寄递时限对标分析指标大全.xlsx` 文件中的 `出口机构表` sheet
2. 根据输入的线路名称，匹配线路名字段，筛选数据
3. 按照收寄时间最长，取出数据中该线路的 N 个机构数据，N 默认 5 个
4. 分析这些机构慢在哪里


## 参数说明

- **第一个参数**：机构名称（必填），如 `"博兴二部"`
- **第二个参数**：线路名称（必填），支持多种格式：
  - `"XX 市 XX 市"` - 直接连接
  - `"XX 市到 XX 市"` - 使用"到"连接
  - `"XX 市 - XX 市"` - 使用"-"连接
  - 程序会自动转换为 `"XX 市 XX 市"` 格式进行匹配
- **第三个参数**：取前 N 个机构（可选），默认为 5

## 使用示例

```
分析出口环节，机构："博兴二部"，线路："北京市上海市"
分析出口环节，机构："博兴二部"，线路："北京市到上海市"
分析出口环节，机构："博兴二部"，线路："北京市 - 上海市"，取前 10 个机构
```

## 分析维度

对筛选出的机构，从以下维度分析慢的原因：

1. **收寄环节**：收寄时长、收寄及时率
2. **运输环节**：运输时长、运输及时率
3. **处理环节**：处理时长、处理及时率
4. **投递环节**：投递时长、投递及时率
5. **全程时限**：全程平均时长、全程及时率

## 输出内容

分析报告包含：
- 线路概况：该线路的机构总数、平均全程时限
- 慢机构列表：收寄时间最长的 N 个机构及其各项指标
- 问题识别：识别这些机构在哪些环节表现较差
- 优化建议：针对问题环节提出改进建议
