# 出口机构下钻分析

## 功能描述

此 skill 用于分析指定出口机构的到达和离开时间分布数据，读取 Excel 数据后找出到达和离开的时间波峰（局部最大值），生成分析总结 PDF。

**输出格式**：
- 自动生成分析结果 PDF：`出口机构下钻分析结果_[机构名称].pdf`
- 分析完成后提示"相关分析已经保存为 PDF 报告"

## 使用场景

当用户需要：
- 分析某出口机构的到达和离开时间分布
- 了解到达和离开的时间波峰
- 找出出口机构的时间分布特征
- 对比到达和离开的时间分布差异

**触发示例**：
- "分析博兴二部出口机构"
- "看看滨州市大同市线路的出口机构到达离开情况"
- "滨州市 - 大同市出口机构下钻分析"
- "分析博兴二部的到达和离开时间波峰"

## 文件位置

- **数据文件**：`../data/寄递时限对标分析指标大全.xlsx`
- **输出文件**：
  - `输出结果/出口机构下钻分析结果_[机构名称].pdf`

## 执行方式

```bash
cd c:\Users\TJY\Desktop\对标 skill\.trae\skills\export-institution-drilldown-analysis
node analyze.js "博兴二部" "滨州市大同市"
```

**参数说明**：
- 第一个参数：机构名称（必填），如 `"博兴二部"`
- 第二个参数：线路名称（必填），如 `"滨州市大同市"`
- 支持多种线路名称格式：
  - `"XX 市 XX 市"` - 直接连接
  - `"XX 市到 XX 市"` - 使用"到"连接
  - `"XX 市 - XX 市"` - 使用"-"连接
  - 程序会自动转换为"XX 市 XX 市"格式进行匹配

**说明**：
- 每一行代表一个出口机构在特定线路上的到达或离开数据
- 时间分布数据为占比（0-1 之间的小数）

- 离开时间分布：0-47 小时，共 48 个数据点

## 分析逻辑

1. **数据筛选**：
   - 根据机构名称筛选
   - 根据线路名称筛选
   - 环节标识为"出口"
   - 到达分拣离开标识为"到达"或"离开"
2. **时间分布提取**：
   - 到达：提取 0-23 小时（24 个数据点）的占比数据
   - 离开：提取 0-47 小时（48 个数据点）的占比数据
3. **局部最大值识别**：找出到达和离开时间分布的局部最大值（峰值点）
4. **峰值分析**：分析每个峰值点的时点和占比
5. **对比分析**：对比到达和离开的时间分布特征

## 实现细节

**分析维度**：
- 到达时间分布（0-23 小时）
- 离开时间分布（0-47 小时）
- 到达峰值识别
- 离开峰值识别
- 到达离开对比分析

**PDF 内容结构**：
1. 机构和线路基本信息
2. 到达时间分布数据及峰值
3. 离开时间分布数据及峰值
4. 到达离开对比分析
5. 分析总结

## 术语说明

- **到达时间分布（0-23 小时）**：邮件在到达后 0-23 小时内各时间点的占比，共 24 个数据点
- **离开时间分布（0-47 小时）**：邮件在离开处理中心 0-47 小时内各时间点的占比，共 48 个数据点
- **波段**：连续时刻有业务量的时间段
- **波峰（峰值）**：一个波段中业务量占比最高的时点
- **峰值时点**：出现波峰的具体时间（小时）
- **峰值占比**：峰值时点对应的业务量占比百分比
- **环节标识**：标识数据属于"出口"环节还是其他环节
- **到达分拣离开标识**：标识数据属于"到达"还是"离开"状态
