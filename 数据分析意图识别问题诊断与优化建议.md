## 四、关键词匹配算法详解

### 4.1 当前算法

```python
def detect_analysis_type(question: str) -> str:
    question_lower = question.lower()
    for analysis_type, keywords in analysis_keywords.items():
        if any(keyword in question_lower for keyword in keywords):
            return analysis_type
    return "chat"  # ❌ 问题：返回 chat 但实际执行了 descriptive
```

### 4.2 算法问题

| 问题 | 描述 | 影响 |
|------|------|------|
| **关键词缺失** | 缺少"对比"、"差异"等核心关键词 | 无法识别对比分析 |
| **匹配逻辑单一** | 简单的字符串包含匹配 | 无法处理复杂语义 |
| **优先级缺失** | 所有关键词同等权重 | 无法处理多意图 |
| **无回退机制** | 未匹配时返回默认值 | 可能执行错误类型 |

### 4.3 匹配流程图

```
用户查询: "对比A与B在XX区域的差异"
    ↓
转小写: "对比a与b在xx区域的差异"
    ↓
遍历 analysis_keywords:
    - correlation: ["相关", "关系", ...] → ❌ 不匹配
    - regression: ["回归", "斜率", ...] → ❌ 不匹配
    - clustering: ["聚类", "分组", ...] → ❌ 不匹配
    - trend: ["趋势", "时间", ...] → ❌ 不匹配
    - anomaly: ["异常", "异常值", ...] → ❌ 不匹配
    - distribution: ["分布", "直方图", ...] → ❌ 不匹配
    - classification: ["分类", ...] → ❌ 不匹配
    - time_series: ["时间序列", ...] → ❌ 不匹配
    - prediction: ["预测", ...] → ❌ 不匹配
    ↓
返回: "chat" 或 "descriptive"  ← ❌ 错误的回退
    ↓
执行: descriptive_analysis()  ← ❌ 用户期望的是对比分析
```

### 5.2 方案二：增强对比分析功能

#### 5.2.1 添加对比分析工具方法

```python
@staticmethod
def comparison_analysis(df: pd.DataFrame, columns: List[str], 
                       compare_columns: List[str] = None,
                       regions: List[str] = None) -> Dict[str, Any]:
    """对比分析"""
    print(f"执行对比分析，列: {columns}")
    result = {
        "analysis_type": "comparison",
        "columns": columns,
        "comparison": {},
        "regions": regions or [],
        "charts": {}
    }
    
    for col in columns:
        if col in df.columns and pd.api.types.is_numeric_dtype(df[col]):
            # 计算基本统计量
            stats = df[col].describe().to_dict()
            
            # 按区域分组计算
            if regions:
                region_stats = {}
                for region in regions:
                    if region in df.columns:
                        region_data = df[df[region] == region][col]
                        region_stats[region] = {
                            "mean": float(region_data.mean()),
                            "std": float(region_data.std()),
                            "min": float(region_data.min()),
                            "max": float(region_data.max())
                        }
                result["comparison"][col] = {
                    "overall": stats,
                    "by_region": region_stats,
                    "differences": _calculate_differences(region_stats)
                }
            else:
                result["comparison"][col] = {"stats": stats}
    
    return result

@staticmethod
def _calculate_differences(region_stats: Dict) -> Dict:
    """计算区域间差异"""
    regions = list(region_stats.keys())
    differences = {}
    
    for i in range(len(regions)):
        for j in range(i + 1, len(regions)):
            r1, r2 = regions[i], regions[j]
            diff = abs(region_stats[r1]["mean"] - region_stats[r2]["mean"])
            differences[f"{r1}_vs_{r2}"] = {
                "absolute_diff": float(diff),
                "relative_diff": float(diff / max(region_stats[r1]["mean"], 0.001))
            }
    
    return differences
```
