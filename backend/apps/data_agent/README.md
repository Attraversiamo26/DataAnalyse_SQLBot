# 数据分析工具使用文档

## 1. 工具介绍

数据分析工具是一个基于Python和大语言模型的智能数据分析系统，能够帮助用户快速分析数据并生成专业的分析报告。

### 主要功能
- **Python代码执行**：安全执行Python数据分析代码
- **智能报告生成**：基于大语言模型生成专业的分析报告
- **数据可视化**：支持多种图表类型的可视化展示
- **灵活的数据源**：支持JSON格式的数据输入

## 2. 系统架构

### 2.1 后端架构
- **代码执行服务**：负责安全执行Python代码
- **报告生成服务**：使用大语言模型生成分析报告
- **API接口**：提供RESTful API接口供前端调用

### 2.2 前端架构
- **数据分析页面**：用于输入数据和分析需求
- **报告查看页面**：用于查看分析结果和报告
- **可视化组件**：用于展示分析结果的图表

## 3. 使用方法

### 3.1 数据分析功能

1. **访问数据分析页面**：在工具选择器中选择"数据分析工具"
2. **输入分析需求**：在"分析需求"文本框中输入您的分析需求，例如"分析这些学生的成绩分布"
3. **输入数据**：在"数据输入"文本框中输入JSON格式的数据
4. **设置依赖包**：在"依赖包"文本框中输入需要的Python依赖包，每行一个
5. **点击"开始分析"**：系统会执行分析并生成报告
6. **查看分析结果**：分析完成后，系统会显示分析结果和生成的报告

### 3.2 报告生成功能

1. **访问报告生成页面**：在数据分析工具中选择"报告生成"标签页
2. **输入用户查询**：在"用户查询"文本框中输入用户的原始查询
3. **输入分析结果**：在"分析结果"文本框中输入Python分析的结果
4. **点击"生成报告"**：系统会生成分析报告
5. **查看生成的报告**：系统会显示生成的报告

## 4. API接口

### 4.1 数据分析接口

**接口地址**：`/api/v1/data-agent/analyze`
**请求方法**：POST
**请求参数**：
```json
{
  "query": "分析这些学生的成绩分布",
  "data": "[{\"name\": \"Alice\", \"age\": 25, \"score\": 85}, {\"name\": \"Bob\", \"age\": 30, \"score\": 90}]",
  "requirements": "pandas"
}
```
**响应格式**：
```json
{
  "code": 0,
  "data": {
    "success": true,
    "result": "{\"total_rows\": 2, \"summary\": {...}, \"columns\": [...]}",
    "report": "本次分析共包含2名学生..."
  }
}
```

### 4.2 报告生成接口

**接口地址**：`/api/v1/data-agent/generate-report`
**请求方法**：POST
**请求参数**：
```json
{
  "user_query": "分析这些学生的成绩分布",
  "analysis_result": "{\"total_rows\": 2, \"summary\": {...}, \"columns\": [...]}"
}
```
**响应格式**：
```json
{
  "code": 0,
  "data": {
    "success": true,
    "report": "本次分析共包含2名学生..."
  }
}
```

## 5. 示例

### 5.1 数据分析示例

**输入**：
- 分析需求："分析这些学生的成绩分布"
- 数据：
```json
[
  {"name": "Alice", "age": 25, "score": 85},
  {"name": "Bob", "age": 30, "score": 90},
  {"name": "Charlie", "age": 28, "score": 75},
  {"name": "David", "age": 35, "score": 95},
  {"name": "Eve", "age": 22, "score": 80}
]
```
- 依赖包："pandas"

**输出**：
- 分析结果：
```json
{
  "total_rows": 5,
  "summary": {
    "age": {"count": 5.0, "mean": 28.0, "std": 4.95, "min": 22.0, "max": 35.0},
    "score": {"count": 5.0, "mean": 85.0, "std": 7.91, "min": 75.0, "max": 95.0}
  },
  "columns": ["name", "age", "score"]
}
```
- 分析报告："本次分析共包含5名学生。在成绩分布方面，分数最低为75分，最高为95分，平均成绩为85分，分数波动数值为7.91。"

### 5.2 报告生成示例

**输入**：
- 用户查询："分析这些学生的成绩分布"
- 分析结果：
```json
{
  "total_rows": 5,
  "summary": {
    "age": {"mean": 28.0, "std": 4.74, "min": 22.0, "max": 35.0},
    "score": {"mean": 85.0, "std": 7.07, "min": 75.0, "max": 95.0}
  },
  "columns": ["name", "age", "score"]
}
```

**输出**：
- 分析报告："本次分析共涉及 5 名学生的数据。成绩方面，平均分为 85 分，最高 95 分，最低 75 分，波动值为 7.07。年龄方面，平均 28 岁，最小 22 岁，最大 35 岁。"

## 6. 常见问题

### 6.1 代码执行失败
- **原因**：Python 环境缺少必要的库
- **解决方案**：在"依赖包"文本框中添加需要的依赖包

### 6.2 报告生成失败
- **原因**：大模型配置错误或网络问题
- **解决方案**：检查大模型配置是否正确，确保网络连接正常

### 6.3 数据格式错误
- **原因**：输入的数据不是有效的JSON格式
- **解决方案**：确保输入的数据是有效的JSON格式

## 7. 性能优化

1. **使用合适的依赖包**：只添加必要的依赖包，减少安装时间
2. **优化数据大小**：对于大规模数据，考虑分批处理
3. **使用高效的Python代码**：避免使用过于复杂的算法
4. **设置合理的超时时间**：根据数据大小和复杂度设置合适的超时时间

## 8. 安全注意事项

1. **代码执行安全**：系统会在隔离的环境中执行Python代码，确保安全性
2. **数据安全**：用户输入的数据不会被存储，分析完成后会被删除
3. **依赖包安全**：只允许安装经过验证的依赖包
